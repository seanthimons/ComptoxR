# Quick Task 260417-hm9: Move environment setup from .onAttach to .onLoad

## Problem
Environment variables (`ctx_burl`, `chemi_burl`, etc.) were only set in `.onAttach()`, which runs only with `library(ComptoxR)`. When using namespace access (`ComptoxR::function()`), `.onAttach()` never runs, leaving environment variables unset and causing API calls to fail.

## Solution
Move server URL setup and defaults from `.onAttach()` to `.onLoad()`, which runs for both `library()` and namespace access.

## Tasks
1. Move environment setup code from `.onAttach()` to `.onLoad()` in `R/zzz.R`
2. Keep startup messages in `.onAttach()` (appropriate only for explicit attachment)
3. Verify fix works with namespace access
