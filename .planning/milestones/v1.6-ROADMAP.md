# Milestone v1.6: Unified Stub Generation Pipeline

**Status:** ✅ SHIPPED 2026-01-30
**Phases:** 10
**Total Plans:** 1

## Overview

This milestone consolidated the stub generation pipeline into a single unified entry point. Previously, `generate_stubs.R` had divergent code paths for different schema types (ct used `openapi_to_spec()`, chemi used `parse_chemi_schemas()`). After this milestone, all schema parsing goes through the same `openapi_to_spec()` pipeline, ensuring consistent behavior for version detection, body extraction, and filtering.
