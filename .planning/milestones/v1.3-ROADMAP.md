**Details:**

Fixed three issues in the resolver-wrapped stub template:
1. Parameter naming: Changed `id_type` to `idType` to match function signature
2. Emptiness check: Changed `nrow(resolved)` to `length(resolved)` for list handling
3. List iteration: Changed tibble row access to `purrr::map()` with element access
