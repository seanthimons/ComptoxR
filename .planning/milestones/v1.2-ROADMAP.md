# Milestone v1.2: Bulk Request Body Type Fix

**Status:** SHIPPED 2026-01-28
**Phases:** 4
**Total Plans:** 3

## Overview

Fixed stub generation to send JSON arrays for bulk POST endpoints, preserving raw text only for `/chemical/search/equal/`. The v1.1 fix for raw text bodies accidentally broke all other bulk POST endpoints by applying newline collapsing to all string_array body types.
