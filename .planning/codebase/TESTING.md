# Testing Patterns

Analysis date: 2026-06-26

## Framework

ComptoxR uses `testthat` edition 3. The default test lane is CRAN-safe: it should pass without a real `ctx_api_key`, live EPA APIs, local database downloads, or cassette recording.

Targeted commands:

```r
testthat::test_file("tests/testthat/test-generic_request.R")
devtools::test(filter = "generic_request")
```

Full-suite commands:

```r
devtools::test()
devtools::check()
covr::package_coverage()
```

## Inventory

The readiness inventory is generated by:

```bash
Rscript dev/unit_test_readiness_audit.R
Rscript dev/unit_test_readiness_audit.R --check-exports
```

The report is `dev/reports/unit_test_readiness_audit.json`. The old `dev/test_manifest.json` file is retired and should be absent; it is not a readiness authority and does not suppress gaps.

## Generated Contract Tests

The active generator is:

```bash
Rscript dev/generate_tests.R --generate
Rscript dev/generate_tests.R --check
Rscript dev/generate_tests.R --dry-run
```

Generated tests are created from exported API wrapper functions, not endpoint slugs. They mock shared request helpers and assert request metadata. Generated files must have the `dev/generate_tests.R` header and must not use VCR.

The generator owns ordinary API-wrapper request-contract coverage. Handwritten tests should remain for shared helpers, algorithms, local-data behavior, generator/audit tooling, and edge cases that need bespoke assertions.

## Mocking

Use `testthat::local_mocked_bindings()` or `testthat::with_mocked_bindings()` for package-local dependencies and HTTP helper boundaries.

Example:

```r
testthat::local_mocked_bindings(
  generic_request = function(...) tibble::tibble(dtxsid = "DTXSID7020182"),
  .package = "ComptoxR"
)
```

Use real package logic for pure data transformation and validation code when possible.

## VCR

VCR is an explicit integration lane. Do not re-record cassettes during routine generated-test validation or CRAN-readiness cleanup.

Intentional recording:

```bash
Rscript dev/rerecord_cassettes.R --record-live --all
Rscript dev/rerecord_cassettes.R --record-live --failures
```

Before accepting cassette changes:

```r
source("tests/testthat/helper-vcr.R")
check_cassette_errors()
check_cassette_safety()
```

Never commit real API keys, tokens, private URLs, or unredacted authorization headers.

## Handwritten Test Scope

Keep handwritten tests for:

- `generic_request()`, `generic_chemi_request()`, `generic_cc_request()`, `generic_cts_request()`, and pagination behavior
- `clean_unicode()` and `safe_tidy_bind()`
- DSS, ECOTOX, and ToxVal install/query/server behavior
- GenRA math, validation, print methods, and offline workflow behavior
- lifestage curation gates and source-backed data integrity
- generated-test pipeline and readiness-audit tooling

Delete handwritten VCR tests that only duplicate wrapper request construction already covered by generated contracts.
