  <action>
Create R/hook_registry.R with:

1. `.HookRegistry <- new.env(parent = emptyenv())` at file top level

2. `load_hook_config()` function (internal, @noRd):
   - Uses `system.file("hook_config.yml", package = "ComptoxR")` to locate config
   - If file exists, calls `yaml::read_yaml(config_path)` and assigns to `.HookRegistry$config`
   - If file missing, assigns empty list: `.HookRegistry$config <- list()`
   - Returns invisible(NULL)

3. `run_hook(fn_name, hook_type, data)` function (internal, @noRd):
   - Looks up `.HookRegistry$config[[fn_name]][[hook_type]]`
   - If NULL (no hooks registered), returns `data` unchanged
   - If hook chain found, iterates over each hook name and calls `match.fun(hook_name)(data)`
   - Use `match.fun()` not `get()` — safer function lookup
   - Returns transformed data after all hooks in chain execute
   - Valid hook_type values: "pre_request", "post_response", "transform"
