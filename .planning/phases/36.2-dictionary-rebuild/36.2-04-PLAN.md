**Section 1: Baseline Schema Check**
- Load `lifestage_baseline.csv` via dual-path pattern (system.file + dev fallback)
- `stopifnot(ncol(baseline) == 13)` (13 columns per cache schema)
- `stopifnot(nrow(baseline) == 139)` (139 terms)
- Print resolved/unresolved/ambiguous counts

**Section 2: Alias CSV Schema Check**
- Load `lifestage_aliases.csv` via dual-path pattern
- `stopifnot(all(c("org_lifestage", "normalized_query") %in% names(aliases)))`
- `stopifnot(nrow(aliases) == dplyr::n_distinct(aliases$org_lifestage))` (no duplicates)

**Section 3: Derivation Schema Check**
- Load `lifestage_derivation.csv` via dual-path pattern
- `stopifnot(ncol(derivation) == 5)`
- Expected columns: source_ontology, source_term_id, harmonized_life_stage, reproductive_stage, derivation_source
- `stopifnot(nrow(derivation) > 57)` (must have grown from original 57)

**Section 4: 100% Coverage Gate (D-07)**
- For each of the 139 baseline terms, check that EITHER:
  - The term has `source_match_status == "resolved"` AND its `(source_ontology, source_term_id)` key exists in derivation
  - OR the term appears in the audit CSV AND has a derivation row with `source_ontology == NA` (unresolvable term)
- Terms with `source_match_status == "ambiguous"` should also have a derivation row
- `stopifnot(nrow(coverage_gaps) == 0)` -- abort with term names if any gaps

**Section 5: No Orphaned Derivation Rows**
- Anti-join derivation against resolved baseline keys (for rows with non-NA source_ontology)
- Orphans = derivation rows whose (source_ontology, source_term_id) never appear in baseline resolved rows
- Allow orphans from the original curated set (derivation_source == "baseline_curated_source_id") since those may have been curated before this re-resolution
- Warn (not abort) if orphans found

**Section 6: Cross-Check Gate**
- Same as validate_36.R Section 2: every resolved baseline key has a derivation partner
- `stopifnot(nrow(gaps) == 0)` with key names on failure

**Section 7: GO:0040007 Contamination Check**
- Same as validate_36.R Section 3
- `stopifnot(sum(baseline$source_term_id == "GO:0040007", na.rm = TRUE) == 0)`

**Footer:**
