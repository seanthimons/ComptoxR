**Function 1: `.eco_lifestage_alias_lookup(org_lifestage)`**

Insert at approximately line 537 (before the OLS4 adapter). Follow the `system.file()` + dev-path fallback pattern from lines 829-838.

```r
#' @keywords internal
.eco_lifestage_alias_lookup <- function(org_lifestage) {
  alias_path <- system.file(
    "extdata", "ecotox", "lifestage_aliases.csv", package = "ComptoxR"
  )
  if (!nzchar(alias_path)) {
    alias_path <- file.path("inst", "extdata", "ecotox", "lifestage_aliases.csv")
  }
  if (!nzchar(alias_path) || !file.exists(alias_path)) {
    return(org_lifestage)
  }
  aliases <- readr::read_csv(alias_path, show_col_types = FALSE)
  match_row <- aliases[aliases$org_lifestage == org_lifestage, ]
  if (nrow(match_row) == 0L || is.na(match_row$normalized_query[[1]])) {
    return(org_lifestage)
  }
  match_row$normalized_query[[1]]
}
```

Key behaviors:
- Returns original `org_lifestage` unchanged if alias file not found or term has no alias
- Uses `system.file()` for installed package, `file.path("inst", ...)` fallback for dev
- No caching (file is tiny at 69 rows)
